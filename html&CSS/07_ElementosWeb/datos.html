Formulario

Los formularios se incluyen dentro de la etiqueta form. En ellos se utlizan los siguientes controles (elementos del formulario que son activos; con los que el usuario puede interactuar).

  • Input text
  • Text ares
  • Checkbox
  • Radio buttons
  • Select / options
  • Botones

La etiqueta form tiene dos atributos importantes

  • Action
  • Method

Los datos por get aparecen en la petición, en la url cuando envíes lo datos

Formulario mínimo

Ejemplo de los elementos minimos de un formulario y uso de los distintos metodos: GET y POST

Get y post sirven para especificar de qué manera se van a enviar los datos que envíe el usuario: En el caso de GET la información se ve reflejada en la URL de la página una vez que se envíe la información, en cambio, usamos POST para cuando la información que recopilamos es secreta POST (información sensible), o no queremos que se descifre desde la URL (Como cuentas bancarias o contraseñas).
